## Week 5-6: 飞书 Bot + 异常预警

### ✅ 已完成

1. **飞书 Bot 完整版** (`feishu-bot/bot.py`)
   - 命令交互：/宏观、/估值、/资金、/情绪、/分析、/预警

2. **命令系统**
   | 命令 | 功能 |
   |------|------|
   | /宏观 [国家] | 宏观经济概览 |
   | /估值 [标的] | 市场/行业估值 |
   | /资金 [类型] | 资金流向 |
   | /情绪 [市场] | 市场情绪 |
   | /分析 [标的] | 四维综合分析 |
   | /预警 列表 | 查看预警设置 |
   | /早报 | 立即获取早报 |
   | /晚报 | 立即获取晚报 |

3. **早晚报模板**
   ```
   📊 市场早报 | 2026-04-07

   • 美联储维持利率不变
   • 纳指 +0.8%
   
   💰 资金信号
   • 北向资金：inflow
   • 两融余额：expanding
   
   📌 四维评分：6.8/10（中性）
   ```

4. **异常预警配置**
   | 指标 | 阈值 | 触发条件 |

   | 单日跌幅 | > 3% | 警示 |
   | 恐贪指数 | < 20 | 机会信号 |

## 使用方法

### 启动 Bot

```bash
cd feishu-bot
cp ../backend/.env.example .env
# 配置飞书 App ID 和 Secret

python run.py
```

### 测试命令

```bash
# 生成早报
python run.py --morning-report

# 生成晚报
python run.py --evening-report

# 测试 Agent
python run.py --test

# 检查预警
python run.py --alerts
```
